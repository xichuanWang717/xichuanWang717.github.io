# 第五章「电影星海」操作说明

## 鼠标模式

进入第五章后，默认使用鼠标探索：

- 拖拽画面：旋转电影星海
- 鼠标滚轮：靠近 / 远离
- 点击气泡：查看电影详情
- 点击气泡时会出现爆炸粒子，表示气泡被“戳破”
- 当下一个气泡被点开时，上一个气泡会恢复到星海中
- 点击详情卡右上角：关闭详情
- 点击「全部」：显示全部电影
- 点击「TOP5」：突出每年票房前五电影
- 点击「重置」：恢复默认视角
- 点击「规则」：查看颜色、大小、高度、光弦、Emoji 和手势含义

## 手势模式

1. 进入第五章「下一帧」。
2. 点击左下角「手势」按钮。
3. 浏览器请求摄像头权限时，选择允许。
4. 状态显示「摄像头：已连接」后即可开始。

手势含义：

- 张手移动：旋转星海
- 捏合：选择气泡并打开电影详情
- 挥手：关闭详情卡
- 双手靠近 / 分开：缩小 / 放大星海

如果摄像头连接失败，页面会自动回到鼠标探索方式。

## 手机 / 触控

如果在手机或触屏设备上观看：

- 单指拖动：旋转星海
- 点击气泡：查看电影详情
- 建议优先使用「全部 / TOP5 / 规则」按钮进行浏览
- 手势识别依赖浏览器摄像头权限，展示时建议使用电脑端更稳定

## 数据编码

- 年份色谱：2018-2025 使用不同气泡颜色
- 类型星座：电影类型决定所在方向和外围标签
- 气泡大小：票房占比
- 气泡高度与光晕：豆瓣评分
- 金色外环：年度 TOP5 影片
